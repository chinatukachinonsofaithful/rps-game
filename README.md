# Rock Paper Scissors Game

A simple, interactive Rock Paper Scissors game built with **HTML, CSS, and JavaScript**, where you challenge the computer and track your wins.

## Live Demo

Play the game online:  
https://rps-game-ten-rouge.vercel.app/

## How to Play

1. Click **Rock**, **Paper**, or **Scissors**.  
2. The AI will randomly pick its choice.  
3. See who wins, loses, or draws.  
4. The scoreboard and game count update automatically.  
5. Click **Reset** to start fresh.

### Rules

- **Rock** beats **Scissors**  
- **Scissors** beats **Paper**  
- **Paper** beats **Rock**  
- Same choice = Draw

## Features

- Live **scoreboard** for player and AI  
- **Game count** tracking  
- **Light/Dark mode** toggle  
- **How to play** instructions  
- **Support the Game** button using Paystack  
- Responsive layout for mobile and desktop

## Technologies Used

- HTML5  
- CSS3 (with CSS variables for theming)  
- JavaScript (ES6+)
