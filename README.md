# Rock Paper Scissors Game

A simple, interactive Rock Paper Scissors game built with **HTML, CSS, and JavaScript**. Challenge the AI and see if you can beat the computer!

## Demo
Open `index.html` in your browser to play the game locally.

## How to Play
1. Click **Rock**, **Paper**, or **Scissors**.  
2. The AI will randomly select its choice.  
3. The result will display whether you **win**, **lose**, or **draw**.  
4. Scores are automatically updated, and the total **game count** is tracked.  
5. Click **Reset** to start over.

### Rules
- Rock beats Scissors  
- Scissors beats Paper  
- Paper beats Rock  
- Same choice = Draw  

## Features
- Interactive **scoreboard** for player and AI  
- Automatic **winner calculation**  
- **Light and dark mode** toggle  
- **Donation support** button using Paystack  
- Fully responsive layout for mobile and desktop  

## Technologies Used
- HTML5  
- CSS3 (variables for theming)  
- JavaScript (ES6+)
